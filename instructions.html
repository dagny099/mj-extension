How to Use Midjourney Image Tracker

This extension helps you bookmark Midjourney images and create a personal gallery.

Where You Can Bookmark Images

  • ✅ Midjourney Create Page (`https://www.midjourney.com/app/create`) – The bookmark icon will appear here.
  • ✅ Midjourney Explore Page (`https://www.midjourney.com/explore`) – The bookmark icon **only appears when an image is expanded.**
  • ❌ Main Explore Grid – The bookmark icon **will NOT appear here.** Click an image first.

How to View Your Bookmarks

Click the extension icon and select "Open Gallery in New Tab".

How to Export Your Bookmarks

  • Click "Download This Page" in the gallery to save an HTML file.
  • Click "Export URLs in Text File" to save just the URLs.
